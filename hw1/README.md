## Домашнее задание 1: описание файлов

#### `complement.py`
Игрушечный скрипт, который получает на вход последовательность и возвращает её reverse-complement и 
значение GC-состава с точностью до третьего знака после запятой.

Пример использования:

```bash
./complement.py --seq ATGACT
AGTCAT
0.333
```
