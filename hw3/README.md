## Домашняя работа 3

#### `parse_traceroute.py`
На вход принимает url, парсит её, доставая имя хоста и, если указан,
порт, а потом запускает `traceroute` с ними в качестве параметров.
В выдаче ищет IP-адреса для каждого хопа. Если выдача для хопа пустая
(`* * *`) - не находится ничего, но иногда бывает, что IP-адресов в
выдаче несколько, и обычно это значит, что пакеты были отправлены на 
разные роутеры для балансировки. Я решила их не выкидывать, а просто 
оставить как элементы одного списка в файле с выдачей.

После для каждого IP находит информацию про него, и если находит - то
пишет в файл `host_name.json`. Не для всех IP что-то удаётся найти, как
минимум потому, что некоторые адреса зарезервированы как приватные. 

*Имя файла решила делать из имени хоста, а не из url, просто для удобства.*

Пример запуска:

```bash
./parse_traceroute.py https://github.com/dkhlebn/FBB-orchestratio
n/blob/main/homeworks/FBBOrch.HW3.ComputerNetworks.Protocols.pdf
```

Файл с выдачей - `github.com.json`.
